# Dating Game (Clean Upload)

This package contains your existing index.html file and an `images/profiles/` folder with a placeholder.png.

## Upload Steps
1. In GitHub, go to your repository.
2. Click "Add file" → "Upload files".
3. Drag **all the files from inside this ZIP** into the upload area.
   - index.html
   - images/profiles/placeholder.png
4. Commit the changes.
5. Go to Settings → Pages → Source: Deploy from branch → Branch: main → Folder: /(root).
6. Save. Wait 1–2 minutes and visit the Pages URL.

You can then add your real images into `images/profiles/` later.
